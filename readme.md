Этот скрипт предназначен для чтения одного или нескольких CSV-файлов,
вычисления среднего значения по столбцу performance для каждого уникального значения в столбце position,
и вывода отсортированного по убыванию результата отчёта.
Скрипт использует библиотеку tabulate для красивого отображения таблицы и поддерживает обработку ошибок при чтении файлов.

Для запуска скрипта укажите один или несколько CSV-файлов и, при необходимости, название отчёта:

python script.py --files file1.csv file2.csv --report my_report
